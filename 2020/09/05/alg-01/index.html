1.1 框架思维

  • 数据结构的存储方式只有两种：数组（顺序存储）、链表（链式存储）。其他多样化的数据结构都是在链表或者数组上的特殊操作。

    数据结构 数组实现 链表实现
    队列 Queue ✓(扩缩容问题) ✓(需要更多空间存节点指针)
    图 Graph ✓(邻接矩阵) ✓(邻接表)
    散列表 Hash table ✓(散列函数把键映射到数组) ✓
    树 Tree ✓(堆) ✓

  • 数组与链表的优缺点对比

    存储方式 优点 缺点
    数组 随机访问 O(1) 扩容 O(N)
    链表 随机访问 O(N) 扩缩容 O(1)

  • 数据结构的基本操作： 遍历 + 访问，两种形式: 线性(迭代for/whie）、非线性（递归）
